## 用自己的话回答

- 什么是特征，什么是标签：特征 (X)：模型用来进行判断的输入信息，也就是用户各项属性；标签 (y)：我们想要预测的标准答案
- 为什么要保留测试集：训练集用来让模型学习规律；测试集全程不参与训练，充当 “陌生新样本”。用来客观检验模型能否泛化到从未见过的数据，排查过拟合
- 为什么83%准确率仍可能没有用：客户流失数据属于不均衡样本，整体流失率仅 16.84%。如果模型无脑预测 都不流失就能拿到约 83.16% 准确率，识别不出流失客户。
